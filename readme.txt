Integration with the RollUp Media Content Suite.

== Description ==

The [RollUp Media Content Suite](http://suites.rollupmedia.com/) will help you to accelerate your business. We enable small and medium-sized publisher to leverage our premium direct sales force delivering outstanding CPMs. Furthermore, we connect Content Creators and Publishers through our editorial content work-flow solution.

== Installation ==
1. Upload the `rollup-media-importer` folder to the `/wp-content/plugins/` directory.
2. Activate the RUM Content Suite plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin by going to the `RUM Content Suite` tab that appears in your admin menu.

== Frequently Asked Questions ==

= WP-Cron =
This plugin use the WP-Cron to syncronize your articles from the RUM Content Suite Platform. So, you need to be sure that this feature is working in your Wordpress installation.
